Sobre mim

Nascido em Pouso Alegre, Minas Gerais, comecei a trabalhar com tecnologia em 2018, ao iniciar minha jornada de desenvolvimento como estagiário. Atuei com várias tecnologias e frameworks até hoje, e sou apaixonado por novas soluções.

Iniciei meus estudos na instituição de ensino UNIVÁS em 2017, com formação prevista para dezembro de 2020. Já no 2º ano, entrei no Instituto Federal do Sul de Minas como estagiário em desenvolvimento web, no 3º ano, entrei no hospital hospital das clínicas Samuel Libânio como estagiário em desenvolvimento web e, para finalizar minha formação, entrei como Estagiário em desenvolvimento web no INATEL.

Minha maior habilidade é na linguagem JavaScript, onde atuo tanto no front-end quando no back-end. Tenho experiência em diversos frameworks, como Angular 2 e ReactJS no front e NodeJS para o back.

Projetos

PsiControl

Esse projeto tem como objetivo o controle de consultas psicológicas dos alunos do curso de Psicologia. É um projeto completo, possui 3 perfis, funcionário, professor e aluno, onde o aluno realiza as consultas e elabora os laudos, o professor valida as consultas e avaliações dos alunos e o funcionário realiza os controle geral.

Poke +

Esse projeto é um projeto mobile, desenvolvido com react native. Tem por finalidade a busca de informações sobre pokemons e itens utilizados em todos os jogos da franquia Pokemon. Foi utilizada uma API pública para busca dos dados.

CDM

Esse projeto é um projeto mobile, desenvolvido com react native. Seu objetivo é realizar a conversão de várias moedas em tempo real, faz o uso de uma API pública para buscar o valor atual das moedas. Foi meu primeiro projeto utilizando esse framework.
